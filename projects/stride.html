StrideML

  • HACK
  • View Github Repo



Stride ML was built for nwHacks 2019. It is an android app that uses accelerometer data to create a biometric profile of an individual, and then sends a notification to a connected smartwatch if a person's phone is stolen by a person with a different walk pattern. Gait profiles are less well-known but potentially more secure methods of biometric authentication. Watch our vlog below!
