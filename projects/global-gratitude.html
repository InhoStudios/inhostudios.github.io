Initialization

January 16, 2018

Global Gratitude is a bot that takes in good vibes and messages and posts them, in an attempt to spread positivity. Currently, it is planned to use the Google Sheets API and the Twitter4J API to tweet out these messages, but it may be easier to start a functional prototype with the JDA API instead. I also plan to implement machine learning to distinguish from those who have legitimately positive messages, and those who want to spread negativity over such a public method.
